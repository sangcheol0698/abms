== Authentication API

=== Login

사용자 계정으로 로그인하고 세션 쿠키를 발급합니다.
로그인 성공 이후의 인증 상태는 세션 기반으로 유지됩니다.
브라우저 클라이언트는 응답의 세션 쿠키를 이후 요청에 함께 전달해야 합니다.

.Request fields
include::{snippets}/auth/login/request-fields.adoc[]

.HTTP request
include::{snippets}/auth/login/http-request.adoc[]

.HTTP response
include::{snippets}/auth/login/http-response.adoc[]

=== Initialize CSRF Token

브라우저 세션용 CSRF 토큰 쿠키를 초기화합니다.
변경 계열 요청(`POST`, `PUT`, `PATCH`, `DELETE`) 전 호출하여 CSRF 쿠키를 준비하는 용도입니다.
